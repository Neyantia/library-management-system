# Podział ról w zespole

## Backend Developer
Odpowiedzialny za:
- implementację backendu w NestJS
- tworzenie API
- projekt i obsługę bazy danych PostgreSQL
- logikę biznesową aplikacji

## Frontend Developer
Odpowiedzialny za:
- implementację interfejsu użytkownika
- komunikację z backendem (API)
- obsługę formularzy i widoków

## UI/UX Designer
Odpowiedzialny za:
- przygotowanie projektu graficznego w Figma
- projektowanie widoków aplikacji
- spójność interfejsu użytkownika

## Team Leader / QA / Dokumentacja
Odpowiedzialny za:
- zarządzanie projektem w Jira
- planowanie zadań i sprintów
- testowanie aplikacji
- zgłaszanie błędów
- przygotowanie dokumentacji projektu

## Organizacja pracy
- zadania są zarządzane w Jira
- kod jest wersjonowany w GitHub
- każdy członek zespołu pracuje na przypisanych zadaniach
- postęp pracy dokumentowany jest poprzez commity i push do repozytorium
